# Egyptian Theme Guide

Tehuti uses an Egyptian-inspired visual voice in the terminal UI.

## Why This Exists

The theme gives the product a distinct identity and makes long sessions more readable and memorable.

## What It Should Influence

- iconography and symbols
- panel titles
- framing language in non-result UI text
- color choices

## What It Must Never Influence

- factual outputs
- tool results
- diagnostics
- error details

In short: style is welcome, fabrication is not.

## Practical Guideline

- Keep symbolic language in headers/prompts/status framing.
- Keep execution summaries literal and evidence-based.
